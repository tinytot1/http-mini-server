## Install

```
$ npm install http-mini-server -g

# Or

$ yarn global add http-mini-server
```

## Usage

```
$ http-mini-server
```

## Options

| options             |                  description |
| :------------------ | ---------------------------: |
| -V, --version       |    output the version number |
| -p, --port <number> | service port (default: 8080) |
| -h, --help          |     output usage information |
